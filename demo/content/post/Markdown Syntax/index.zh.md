## 引用块

引用块元素代表从其他来源引用的内容，可以选择带有来自于 `footer` 或 `cite` 元素的引用说明，并可选择带有行内修改，如注释和缩写。

### 无归属的引用块

> 这是一个没有任何特定归属的引用块示例。
> **注意** 你可以在引用块中使用 *Markdown 语法*。

### 带归属的引用块

> 不要通过共享内存来通信，而要通过通信来共享内存。<br>
> — <cite>Rob Pike[^1]</cite>

[^1]: 上述名言摘自 Rob Pike 在 2015 年 11 月 18 日 Gopherfest 期间的 [演讲](https://www.youtube.com/watch?v=PAAkCSZUG1c)。

## 表格

表格不是 Markdown 核心规范的一部分，但 Hugo 原生支持它们。

   | 姓名  | 年龄 |
   | ----- | --- |
   | Bob   | 27  |
   | Alice | 23  |

### 表格内的行内 Markdown

| 斜体      | 粗体     | 代码   |
| --------- | -------- | ------ |
| *斜体*    | **粗体** | `code` |

| A                                                        | B                                                                                                             | C                                                                                                                                    | D                                                 | E                                                          | F                                                                    |
| -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------- | ---------------------------------------------------------- | -------------------------------------------------------------------- |
| 这是一个很长的单元格内容，主要用于测试表格在不同屏幕尺寸下的响应式表现。 | 这是另一列的内容，同样包含了大量的文字，以确保表格布局能够正确处理长文本。 | 第三列的内容也不短，看看它会如何影响周围的列宽分配以及整体的表格结构。 | 短文本测试 | 中等长度的文本测试，稍微长一点 | 最后的总结性文本，填满表格。 |

## 代码块

### 使用反引号的代码块

```html
<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>Example HTML5 Document</title>
</head>
<body>
  <p>Test</p>
</body>
</html>
```

### 缩进四个空格的代码块

    <!doctype html>
    <html lang="en">
    <head>
      <meta charset="utf-8">
      <title>Example HTML5 Document</title>
    </head>
    <body>
      <p>Test</p>
    </body>
    </html>
