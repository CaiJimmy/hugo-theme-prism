云原生技术使组织能够在现代、动态的环境（如公有云、私有云和混合云）中构建和运行可扩展的应用程序。

<!--more-->

## 微服务

将单体应用程序分解为通过定义良好的 API 进行通信的更小的独立服务。

## 容器

容器（如 Docker）将代码及其所有依赖项打包，以便应用程序可以从一个计算环境快速可靠地运行到另一个计算环境。

## CI/CD

持续集成和持续交付允许频繁且可靠的代码更改。
